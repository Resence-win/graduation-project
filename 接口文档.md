* SpringBoot 3.x
* MyBatis-Plus
* OpenGauss

接口地址：

```

[http://localhost:8080/api](http://localhost:8080/api)

````

# 十二、数据统计模块

## 图表展示通用说明

本模块所有统计数据均支持通过ECharts进行可视化展示，前端可根据实际需求选择合适的图表类型。

### ECharts集成说明

**安装依赖**

```bash
npm install echarts
```

**基本使用**

```javascript
import * as echarts from 'echarts';

const chartDom = document.getElementById('chart');
const myChart = echarts.init(chartDom);
const option = {
  // 图表配置项
};
myChart.setOption(option);
```

### 图表类型选择指南

| 数据类型 | 推荐图表类型 | 适用场景 |
|---------|------------|---------|
| 时间序列数据 | 折线图、面积图 | 消费趋势、充值趋势等 |
| 分类对比数据 | 柱状图、横向柱状图 | 用户排行、商户排行等 |
| 占比数据 | 饼图、环形图 | 商户收入占比等 |
| 多维度数据 | 雷达图、散点图 | 用户画像分析等 |

### 通用图表配置

**响应式设计**

```javascript
window.addEventListener('resize', function() {
  myChart.resize();
});
```

**主题切换**

```javascript
// 亮色主题
const myChart = echarts.init(chartDom, 'light');

// 暗色主题
const myChart = echarts.init(chartDom, 'dark');
```

**数据加载动画**

```javascript
myChart.showLoading({
  text: '加载中...',
  color: '#409EFF',
  textColor: '#000',
  maskColor: 'rgba(255, 255, 255, 0.8)'
});

// 数据加载完成后
myChart.hideLoading();
```

### 图表导出功能

```javascript
option = {
  toolbox: {
    feature: {
      saveAsImage: {
        title: '保存为图片',
        pixelRatio: 2
      },
      dataView: {
        title: '数据视图',
        readOnly: true
      },
      magicType: {
        type: ['line', 'bar'],
        title: {
          line: '切换为折线图',
          bar: '切换为柱状图'
        }
      }
    }
  }
}
```

### 数据格式规范

所有图表数据接口统一返回JSON格式：

```json
{
  "code": 0,
  "msg": "success",
  "data": [
    {
      "字段1": "值1",
      "字段2": "值2"
    }
  ]
}
```

### 图表性能优化

1. **大数据量处理**
   - 使用dataZoom组件实现数据缩放
   - 采用抽样或聚合策略
   - 使用WebGL渲染（echarts-gl）

2. **动画优化**
   - 减少不必要的动画效果
   - 设置合理的动画时长
   - 使用渐进式渲染

3. **内存管理**
   - 及时销毁不再使用的图表实例
   - 避免重复创建图表实例
   - 使用dispose()方法释放资源

## 1. 消费统计分析

GET

```
/api/stat/consume
```

请求参数

```
start_date
end_date
```

数据库逻辑

```
SELECT DATE(consume_time) as date, SUM(amount) as total_amount
FROM consume_record
WHERE consume_time BETWEEN ? AND ? AND is_deleted=0
GROUP BY DATE(consume_time)
ORDER BY date
```

返回示例

```json
{
  "code": 0,
  "msg": "success",
  "data": [
    {
      "date": "2024-03-01",
      "total_amount": 12500.50
    },
    {
      "date": "2024-03-02",
      "total_amount": 13200.80
    },
    {
      "date": "2024-03-03",
      "total_amount": 11800.30
    }
  ]
}
```

图表展示说明

**图表类型：折线图**

**ECharts配置示例**

```javascript
option = {
  title: {

  },

  },

  },
  yAxis: {
    type: 'value',
    name: '消费金额(元)'
  },
  series: [{

    smooth: true,
    itemStyle: {
      color: '#409EFF'
    },
    areaStyle: {
      color: {
        type: 'linear',
        x: 0,
        y: 0,
        x2: 0,
        y2: 1,

          offset: 0, color: 'rgba(64, 158, 255, 0.3)'
```

**图表展示效果**

- 横轴：日期
- 纵轴：消费金额
- 展示指定时间段内的每日消费趋势
- 支持鼠标悬停查看具体日期的消费金额
- 支持缩放和平移查看历史数据

**数据格式要求**

- date：日期字符串，格式为YYYY-MM-DD
- total_amount：消费金额，保留两位小数
- 数据按日期升序排列

**可选图表类型**

- 折线图：展示消费趋势变化
- 柱状图：展示每日消费对比
- 面积图：展示消费累积趋势

## 2. 用户消费排行

GET

```
/api/stat/user-rank
```

请求参数

```
start_date
end_date
limit
```

数据库逻辑

```
SELECT a.card_id, SUM(cr.amount) as total_amount
FROM consume_record cr
JOIN account a ON cr.account_id = a.id
WHERE cr.consume_time BETWEEN ? AND ? AND cr.is_deleted=0 AND a.is_deleted=0
GROUP BY a.card_id
ORDER BY total_amount DESC
LIMIT ?
```

返回示例

```json
{
```
