# 校园一卡通系统 API 接口文档

基于 OpenGauss 的校园一卡通系统数据库设计

后端框架：

* SpringBoot 3.x
* MyBatis-Plus
* OpenGauss

接口地址：

```

[http://localhost:8080/api](http://localhost:8080/api)

````

# 二、认证模块

## 1. 管理员登录

POST

```
/api/admin/login
```

请求参数

```json
{
  "username": "admin",
  "password": "123456"
}
```

数据库逻辑

```
SELECT * FROM admin_user
WHERE username=? AND password=? AND is_deleted=0
```

返回示例

```json
{
  "code": 0,
  "msg": "登录成功",
  "data": {
    "id": 1,
    "username": "admin",
    "role": "SUPER_ADMIN"
  }
}
```

# 十二、数据统计模块

## 1. 消费统计分析

GET

```
/api/stat/consume
```

请求参数

```
start_date
end_date
```

数据库逻辑

```
SELECT DATE(consume_time) as date, SUM(amount) as total_amount
FROM consume_record
WHERE consume_time BETWEEN ? AND ? AND is_deleted=0
GROUP BY DATE(consume_time)
ORDER BY date
```

---

## 2. 用户消费排行

GET

```
/api/stat/user-rank
```

请求参数

```
start_date
end_date
limit
```

数据库逻辑

```
SELECT a.card_id, SUM(cr.amount) as total_amount
FROM consume_record cr
JOIN account a ON cr.account_id = a.id
WHERE cr.consume_time BETWEEN ? AND ? AND cr.is_deleted=0 AND a.is_deleted=0
GROUP BY a.card_id
ORDER BY total_amount DESC
LIMIT ?
```

---

## 3. 商户收入排行

GET

```
/api/stat/merchant-rank
```

请求参数

```
start_date
end_date
limit
```

数据库逻辑

```
SELECT merchant_id, SUM(amount) as total_amount
FROM consume_record
WHERE consume_time BETWEEN ? AND ? AND is_deleted=0
GROUP BY merchant_id
ORDER BY total_amount DESC
LIMIT ?
```
