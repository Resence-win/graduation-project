# 校园一卡通系统 API 接口文档

基于 OpenGauss 的校园一卡通系统数据库设计

后端框架：

* SpringBoot 3.x
* MyBatis-Plus
* OpenGauss

接口地址：

```

[http://localhost:8080/api](http://localhost:8080/api)

````

---

# 一、统一返回格式

```json
{
  "code": 0,
  "msg": "success",
  "data": {}
}
````

# 二、认证模块

## 1. 管理员登录

POST

```
/api/admin/login
```

请求参数

```json
{
  "username": "admin",
  "password": "123456"
}
```

数据库逻辑

```
SELECT * FROM admin_user
WHERE username=? AND password=? AND is_deleted=0
```

返回示例

```json
{
  "code": 0,
  "msg": "登录成功",
  "data": {
    "id": 1,
    "username": "admin",
    "role": "SUPER_ADMIN"
  }
}
```
