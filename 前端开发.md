## 八、Axios封装（必须做）

### utils/request.js

```javascript
import axios from 'axios'
import { ElMessage } from 'element-plus'
import router from '@/router'

const request = axios.create({
  baseURL: 'http://localhost:8080/api',
  timeout: 10000
})

// 请求拦截器
request.interceptors.request.use(
  config => {
    const token = localStorage.getItem('token')
    if (token) {
      config.headers['Authorization'] = `Bearer ${token}`
    }
    return config
  },
  error => {
    return Promise.reject(error)
  }
)

// 响应拦截器
request.interceptors.response.use(
  response => {
    const res = response.data
    
    if (res.code === 0) {
      return res
    } else {
      ElMessage.error(res.msg || '请求失败')
      return Promise.reject(new Error(res.msg || '请求失败'))
    }
  },
  error => {
    console.error('请求错误:', error)
    
    if (error.response) {
      switch (error.response.status) {
        case 401:
          ElMessage.error('未授权，请重新登录')
          localStorage.removeItem('token')
          localStorage.removeItem('user')
          router.push('/login')
          break
        case 403:
          ElMessage.error('拒绝访问')
          break
        case 404:
          ElMessage.error('请求地址不存在')
          break
        case 500:
          ElMessage.error('服务器错误')
          break
        default:
          ElMessage.error(error.response.data.message || '请求失败')
      }
    } else if (error.request) {
      ElMessage.error('网络错误，请检查网络连接')
    } else {
      ElMessage.error('请求失败')
    }
    
    return Promise.reject(error)
  }
)

export default request
```

- 前后端分离架构便于维护和扩展
- Element Plus提供统一的UI风格
- ECharts实现数据可视化展示

#### 系统特色功能
- 完整的权限控制机制
- 良好的用户体验设计
- 响应式布局适配不同设备
- 统一的错误处理和提示

### 2. 可能的问题与回答

#### Q1: 为什么选择Vue3而不是Vue2？
A: Vue3提供了更好的性能、更小的体积、更好的TypeScript支持，Composition API让代码组织更灵活，逻辑复用更方便。

#### Q2: 如何保证前端代码的质量？
A: 使用ESLint进行代码检查，遵循Vue3官方风格指南，使用Git进行版本控制，编写单元测试，定期进行代码审查。

#### Q3: 如何处理大数据量的渲染性能问题？
A: 使用分页加载、虚拟滚动、懒加载等技术，合理使用computed缓存计算结果，避免不必要的重新渲染。

#### Q4: 如何实现前后端的权限控制？
A: 前端使用路由守卫和按钮级权限控制，后端使用JWT进行身份认证，前后端双重保证系统安全。

#### Q5: ECharts图表如何实现数据可视化？
A: 通过调用后端统计接口获取数据，使用ECharts的配置项设置图表类型、样式、交互等，实现数据的可视化展示。
